### 文件
#### config.js 配置按钮的点击事件
#### layout.xml 界面
#### registerViewListeningEvent.js 注册按钮点击事件
#### main.js 入口文件

### 打包
1. 在scriptConfig中修改xmlConfigPath, 这是用于替换xml文件中的变量
2. 在根目录下执行 `webpack`, 打包后的文件在`dist`目录下
3. 不明白的可以参考`https://github.com/snailuncle/webpack-autojs`
